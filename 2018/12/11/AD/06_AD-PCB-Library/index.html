一、简易按键PCB Library绘制

1.1 新建文件找到零点

File_>New_>Library_>PCB Library，快捷键 Ctrl+End 可以直接找到零点

1.2 Pad参数配置

Place Pad 放置焊盘，其中30mil指的是内径，60mil指的是外径。
shape可以更改焊盘形状。
Layer可以设置焊盘在哪一层，其中Multi-Layer是指的从顶层一直穿过极板到达底层。因为这个是要做按键的PCB，而按键的引脚肯定是要从板子穿过去的。
Electrical Type ：Load 负载；Terminator 终端；Source 源。
在此界面中可以用 Ctrl+Q 来进行英制单位和米制单位的切换。

选择Top Overlay 丝印层（顶层上面的层）
4个过孔，8个元件（4个过孔，4条线）

三、利用封装向导绘制PCB Library

3.1 常用封装介绍及选择

如果是手工操作是比较麻烦，可以用封装向导来做，比较方便。
Tools_>Component Wizard


常见封装类型

  • Ball Grid Arrays(BGA) BGA球栅阵列封装
    球栅阵列（Ball Grid Array，简称BGA）封装技术为应用在集成电路上的一种表面黏着技术，此技术常用来永久固定如微处理器之类的的装置。BGA封装能提供比其他如双列直插封装（Dual in-line package）或四侧引脚扁平封装（Quad Flat Package）所容纳更多的接脚，整个装置的底部表面可全作为接脚使用，而不是只有周围可使用，比起周围限定的封装类型还能具有更短的平均导线长度，以具备更佳的高速效能 。
    球栅阵列（Ball Grid Array,简称BGA）封装是在封装体基板的底部制作阵列，焊球作为电路的I/O端与印刷线路板（PCB）互接。采用该项技术封装的器件是一种表面贴装型器件。
    BGA(Ball Grid Array),简称BGA,译为球形触点阵列封装，也可译为“球栅阵列”或“网络焊球阵列”和“球面阵”等等。它是在基板的背面按阵列方式制出球形触点作为引脚，在基板正面装配LSI（部分BGA芯片与引出端在基板同一面）是多引脚LSI用的一种表面贴装型封装。

  • Capacitors 电容封装

  • Diodes 二极管封装
  • Dual In-line Packages(DIP) DIP双列直插封装
  • Edge Connectors 边连接器封装

  • Leadless Chip Carriers(LCC) LCC无引线芯片载体封装
    LCC封装，Leadless Chip Carriers，的形式是为了针对无针脚芯片封装设计的，这种封装采用贴片式封装，它的引脚在芯片边缘地步向内弯曲，紧贴芯片，减小了安装体积。但是这种芯片的缺点是使用时调试和焊接都非常麻烦，一般设计时都不直接焊接到印制线路板上，而是使用PGA封装的结构的引脚转换座焊接到印制线路板上，再将LCC封装的芯片安装到引脚转换座的LCC结构形式的安装槽中，这样的芯片就可随时拆卸，便于调试 。
  • Pin Grid Arrays(PGA) PGA 引脚网格阵列封装
    PGA封装，英文全称为(Pin Grid Array Package)，中文含义叫插针网格阵列封装技术，由这种技术封装的芯片内外有多个方阵形的插针，每个方阵形插针沿芯片的四周间隔一定距离排列，根据管脚数目的多少，可以围成2～5圈。安装时，将芯片插入专门的PGA插座。

  • Quad Packs(QUAD) QUAD四边引出扁平封装
  • Resistors 电阻封装
  • Small Outline Packages(SOP) SOP小尺寸封装

  • Staggered Ball Grid Arrays(SBGA) SBGA交错球栅阵列封装
  • Staggered Pin Grid Arrays(SPGA) SPGA交错引脚网格阵列封装

由于芯片手册显示的是SOP封装，所以在此选择SOP

3.2 根据datasheet来配置PCB

根据datasheet修改焊盘长宽。宽度可以取最大值（16mil），此芯片封装的引脚长度最大是28mil，如果是机器焊接，则无所谓，但如果是手工，那就尽量选大一点。比如改成80mil（1mil=0.0254mm）

这样就设置好了一个焊盘的参数。
由datasheet可知，焊盘中心的距离是26mil，两列焊盘中心的距离
（197+189）/2=192mil (28+16)/2=22mil （122+114）/2=118mil，(192-118)/2-22=15mil但之前将焊盘长度加到了80mil，(15+80/2+118/2)*2=228mil，所以两排焊盘的间距是228mil。或者，原本是192-22=170mil，增加了80-22=58mil，170+58=228mil。

然后再设置外框的线条宽度，默认是10mil。然后再设置焊盘的个数。

由于这两点之间的距离是192-22*2=148mil


实测也是148mil，所以引脚可以焊上，且留出了多余的焊盘，供电烙铁上锡。
