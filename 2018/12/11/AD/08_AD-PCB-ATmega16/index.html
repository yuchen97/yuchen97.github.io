08_AD-PCB-ATmega16

发表于 2018-12-11 | 更新于 2018-12-22 | 分类于 Altium Designer

一、将Schematic导入到PCB

将原理图导入到PCB有多种方法
可以直接New_>PCB，但是这种方法创建的PCB是默认设置的，比如PCB的大小等都不可进行修改，有一定的局限性。
或者右下角，System_>Files_>New from template_>PCB Board Wizard.
PCB Board Wizard_>Imperial(英制)，Metric(米制)_>Custom(可以自己设定参数)。

配置完之后，即可生成相应的PCB。
注：在从原理图导入到PCB的时候，不要忘了把原理图和PCB放在一个Project里，否则Updata会显示灰色。

二、便捷功能

1.在SCH_>Board Insight中按 shift+X可以查看 component的信息；
2.shift+M 放大镜；
3.Edit中的Slice Tracks可以对PCB的线进行切割，相当于原理图中的Break Wires；
4.同时Edit中的Align也可以对元器件进行排序；
5.在PCB中如果想连续选中多个元器件可以按住 shift 来进行操作；
6.原点的位置可以用Edit_>Origin_>Set来操作；
7.在PCB中的J和G都可以有快捷操作；
8.在PCB中按下L可以弹出来层参数的配置

9.比如在PCB中的某个地方设置位置标识J_>Set Location Marks，可以在任意位置设置，然后按 下J_>Location Marks 可以瞬间找到相应的位置。
10.将板子翻转 View_>Flip Boards
11.屏幕刷新 View_Refresh 或者直接按End
12.按下数字3可以切换到3D模式或者 View_>3D Layout Mode
在3D状态下，按住shift ，通过鼠标右键可以对其进行旋转操作。
13.在右下角的PCB_> 3D Visualization的界面中，按下左键可以进行旋转，按下右键可以进行缩放操作。
14.在3D状态下，View里面可以进行90°（按下数字9），0°（按下数字0）等旋转位置的快捷操作。
15.如果想隐藏Component之间或者所有的连线，可以用View_>Connections进行操作。
16.View里面也可以对单位进行切换 Toggle Units快捷键是Q
17.Design_>Board Shape可以对板子的形状进行设置。
18.在Design中Layer Stack Manager可以重新插入一层板子

19.Tools里面Cross Probe选中相应的元件之后，可以快速的在PCB和原理图中进行切换。把这个和Windows下的Title相结合更好用。

20.注意下图的右上角，相当于一个过滤器，输入相应的东西之后，高亮，其他暗色。

21.如果只看单层，可右键单击此层，选中Layer Set_>Signal Layer。或者用 shift+S
按小键盘的“+”或者“-”可以进行单层之间的切换
22.放置元器件时，要想下一个元器件自动弹出来，可以选中相应的Component然后Tools_>Component Placement_>Reposition Selected Components，就ok了。
23.在建立Room之后，点击Tools_>Component Placement_>Arrange Within Room，可以把元器件自动的全部放入Room里面。
24.自动布线在Tools_>Component Placement_>Auto Placer。
25.选中相应的元器件，双击可以进行层的操作，比如原来在顶层，把它放置到底层。或者在拖动元件的过程中，按L，可以在层之间上下切换。
26.这个是布线结束。比如不选中，1和2相连，用引线把他们从1到2连接时，引线还有，但选中之后，连接完2，引线消失。具体说也说不明白，实际用的时候就知道了。

这个是不选中

这个是选中

26.其中Ctrl+Shift+Space可以更改走线模式（45°，圆之类的 ）
27.双击这里可以弹出来更改快捷键的界面

28.shift+R 可以更改Toggle Routing Mode
29.在拖动引线的时候按快捷键提示菜单 “Tab上面的那个键”
30.在布线的时候按住 shift+W 可以快速更改线宽。
31.一般画不同层的线的时候，直接点击 Tab上面的那个键，可以快速切换。
32.选中焊盘，按下P，找到Interactive Multi-Routing，可以一次性画多条线。选中多个焊盘按 shift。

33.如果想改元器件的标签大小，则右键_>Find Similar Objects.


34.如果想隐藏注释，则 右键_>Find Similar Objects_>Object find:Component进行操作。
35.高亮的颜色在 Tools_>Preferences_>Board Insight Display_>Live Highlighting
36.这里可以进行透明度的设置

37.键盘按下S，选择Touching Line，然后移动鼠标会出现一条直线，在这条直线上的都会被选中。

三、规则设置

因为这个板子只需要一种Rules，所以可以把Room删除掉。如果一个板子有不同的Rules，则在每个Rule上加上 Room。
在Keep-Out Layer定义边界，即禁止布线区，相当于定了板子的边缘。选中Keep-Out Layer直接进行画线操作就可以。

四、引线

但在实际操作时遇到了一些问题，比如R1的另一端居然没有和任何一个引脚有连线，到时候画线的时候肯定没法连。而且我在Schematic中这个引脚明明和VCC_5V相连了，而且导入PCB的时候没有错误，也没有警告。


所以在此可以双击这个没有连线的焊盘，将NET和相应的引脚相连即可。


这样即完成了PCB的布局

注：放置的时候尽量把引脚放置到格点处，方便布线。

五、放置铜柱

铜柱的放置，可以直接用焊盘。铜柱的孔径一般是3.3mm，


双击焊盘，将四个焊盘锁定。这样它们位置就固定了。或者选中右键Component Locked

画完板子之后，如果对板子的大小进行标注，可以点


在测量的时候按下 Tab 可以调节参数，现在显示的76.20mm

六、Unions

比如我想同时移动这两个元器件，而且保持元器件的相对位置不变。如果直接选中再移动，可能在选中或者移动的过程中出问题，所以选中之后点击右键_>Unions_>Create Union from Selected Objects，然后就可以把他们作为一个整体来进行移动了。如果想把他们再分开，同样的操作，只是点击Unions之后点Break。

七、信号包地

对重要信号线进行包地处理
假设白色上面的这条线是重要信号线，若对其进行包地处理，首先要选中此Net，要么是一段一段的选（按住shift），但如果线段较多，可用另一种方法。按S_Net，然单击待选择的线。选中之后点击Tools_>Outline Selected Objects


双击包线，将其连接到AGND。
出现绿色报错，原以为是线间距问题，后来发现，默认宽度是10mil，而包线宽是8mil。选中将其改成10mil即可。不过改完之后还是会报错，因为线宽增加了，间距减小了。于是我找到Design_>Rules，对标准进行修改。将线的最小设置为8mil，最大为10mil。因为之前最大和最小都是10mil。


八、铺铜操作

主要是让板子上产生一个大面积的地，将铜和地相连，给地提供一个比较低的阻抗，保证其有一个稳定的电势。铜箔一般Connect to Net GND。

覆铜之后可以多加一些过孔。

九、放置泪滴

Tools_>Teardrops.。增加了导线和圆弧的牢固程度。如果不放置，GND上面的那个弧度将变成矩形。
