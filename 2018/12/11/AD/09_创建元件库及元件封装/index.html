09_创建元件库及元件封装

Posted on 2018-12-11 | Edited on 2019-03-21 | In Altium Designer
Symbols count in article: 4.8k | Reading time ≈ 4 mins.

一、绘制库元件

1.1 绘制库元件的原理图符号

  • File——>New——>Library——>Schematic Library，启动原理图库文件编辑器，并创建一个新的原理图库文件，命名为NewLib.SchLib；
  • 为新建的库文件原理图符号命名
    在创建了一个新的原理图库文件的同时，系统已自动为该库添加了一个默认原理图符号名为Component-1的库文件，打开SCH Library面板可以看到，两种方法对该库文件，进行重新命名。
    • 单击原理图符号绘制工具栏的创建新元件按钮 ，则弹出原理图符号名称对话框，可以在此会话框内输入自己想要的绘制的库文件名称；
    • 在SCH Library面板上，直接单击原理图符号名称框下面的 Add 按钮，也会弹出同样的原理图符号名称对话
  • 单击原理图符号绘制工具栏中放置矩形按钮，则光标变成十字形状，并附有一个矩形符号；
  • 两次单击鼠标左键，在编辑窗口绘制一个矩形。

1.2 放置引脚

  • 单击原理图符号绘制工具栏中的放置引脚按钮，则光标会变成十字形状，并附有一个引脚符号；
  • 移动引脚到矩形边框处，单击左键完成放置；
    放置引脚时，一定要保证具有电气特性的一端，即带有 “×”的一端朝外，可通过在放置引脚时按空格键旋转实现。

  • 在放置引脚时按下Tab，或者双击已放置的引脚 ，系统将会弹出元件引脚属性对话框，完成各项设置；

    • Display Name(显示名称)：设置库元件引脚的名称；
    • Designator(标识)：设置库元件引脚的编号，应与实际的引脚编号相对应；
    • Electrical Type(电气类型)：设置库元件引脚的电气特性。Input(输入)、IO(输入输出)、Output(输出)、OpenCoiiector(打开集流器)、Passive(中性的)、Hiz(脚)、Emitter(发射器)和Power(激励)8个选项。Passive(中性的)表示不设置电气特性；

    • Hide(隐藏管脚)：设置引脚是否为隐藏引脚，若勾选，则引脚将不会显示出来。此时，应在右侧的Connect To文本框中输入与该引脚连接的网络名称；

  • 按照同样的操作，完成其余引脚设置。

1.3 编辑元件属性

(1) 双击SCH Library 面板原理图符号名称栏中的库文件名称，弹出如图对话框；

  • Default Designator(默认符号） : 默认库元件标号， 即把该元件放置到原理图文件中时，系统最初默认显示的元件标号。 这里设置为 “U?”，并勾选右侧的“visible(可用)” 复选框， 则放置该元件时，序号“U? “ 会显示在原理图上;
  • DefauIt comment(元件) 下拉列表框:用于说明库元件型号。勾选右侧的 “Visible （可见）” 复选框， 则放置该元件时名称会显示在原理图上;
  • Description（描述）：用于描述库元件功能;
  • Type（类型）下拉列表框：库元件符号类型，可以选择设置。这里采用系统默认设置 Standard（标准）;
  • Library Link(元件库线路）：库元件在系统中的标识符;
  • Show All pins On Sheet（Even if Hidden ） （在原理图中显示全部引脚）复选框 ： 勾选该复选框后，在原理图上会显示该元件的全部引脚;
  • Lock Pins （锁定引脚）复选框，勾选该复选框后，所有的引脚将和库元件成为一个整体，不能在原理图上单独移动引脚。建议用户勾选该复选框 ，这样对电路原理图的绘制和编辑会有很大好处，以减少不必要的麻烦;
  • 在 “parameters” 列表框中，单击Add按钮，可以为库元件添加其他的参数;
  • 在 “Models（模型）” 列表框中， 单击Add按钮，可以为该库元件添加其他的模型，如 PCB封装模型、信号完整性模型、仿真模型、PCB 3D模型等;
  • 单击左下角的Edit Pins按钮，则会打开元件引脚编辑器，可以对该元件所有引脚进行一次性的编辑设置。

(2) 设置完毕，关闭对话框；
(3) P+T，或单击原理图符号绘制工具栏中的放置文本字符按钮”A”,光标会变成十字形状，并带有一个文本字符串；
(4) 移动光标到原理图符号中心位置处，按下Tab键或双击字符串，弹出文本字符串注释对话框输入要输入的文本。

三、创建PCB元件库及封装

3.1 封装概述

封装是指安装半导体集成电路芯片用的外壳，它起着安放、固定、密封、保护芯片和增强电热性能的作用，也是沟通芯片内部与外部电路的桥梁。芯片的封装在PCB板上通常表现为一组焊盘、丝印层上的边框及芯片的说明文字。焊盘是封装中最重要的组成部分，用于连接芯片的引脚，并通过印制板上的导线来连接印制板上的其他焊盘，进一步连接焊盘所对应的芯片引脚，完成电路板的功能。在封装中，每一个焊盘都有唯一的标号，以区别于封装中的其他焊盘。丝印层上的边框和文字说明主要起指示作用，指明焊盘组所对应的芯片，方便印制板的焊接。焊盘的形状和排列是封装的关键组成部分，确保焊盘的形状和排列正确才能正确地建立一个封装。对于安装有特殊要求的封装，边框也需要绝对正确。

3.2 常用封装介绍

总体上讲， 根据元件采用安装技术的不同 ， 可分为插入式封装技术（Through Technology, THT 〕 和表贴式封装技术 （surface Mounted Technology, SMT ）。
插入式封装元件安装时， 元件安置在板子的一面， 将引脚穿过PCB 板焊接在另一面上。 插入式元件需要占用较大的空间，并且要为每只引脚钻一个孔，所以它们的引脚会占据两面的空间，而且焊点也比较大。但从另一方面来说，插入式元件与PCB连接较好，机械性能好。例如，排线的插座、接囗板插槽等类似的界而都需要一定的耐压能力，因此，通常采用THT封装技术。
表贴式封装的元件，引脚焊盘与元件在同一面。表贴元件一般比插入式元件体积要小，而且不必为焊盘钻孔，甚至还能在PCB板的两面都焊上元件。因此，与使用插入式元件的PCB比起来， 使用表贴元件的PCB板上元件布局要密集很多， 体积也就小很多。 此外， 表贴封装元件也必插入式元件要便宜一些，所以现今的PCB上广泛采用表贴元件。
元件封装可以大致分成以下种类。

  • BGA( Ball Grid Array）： 球栅阵列封装。 因其封装材料和尺寸的不同还细分成不同的 BGA 封装， 如陶瓷球栅阵列封装CBGA、 小型球栅阵列封装 μBGA等。
    • PGA（Pin Grid Array）： 插针栅格阵列封装技术。 这种技术封装的芯片内外有多个方阵形的插针， 每个方阵形插针沿芯片的四周间隔一定距离排列， 根据管脚数目的多少， 可以围成2~5圈。 安装时， 将芯片插入专门的PGA插座。 该技术一般用于插拔操作比较频繁的场合之下， 如个人计算机CPU。
    • QFP（ Quad Flat Package ）： 方形扁平封装， 为当前芯片使用较多的一种封装形式。
    • PLCC（Plastic Leaded Chip carrier）： 有引线塑料芯片载体。
    • DIP （Dual In-line Package）：双列直插封装。
    • SIP（Single In-line Package）： 单列直插封装。
    • SOP（Small Out-line Package）： 小外形封装。
    • SOJ（small Out-line J-LeadedPackage）：J形引脚小外形封装。
    • CSP（Chip Scale Package）：芯片级封装，较新的封装形式，常用于内存条中。在CSP的封装方式中，芯片是通过一个个锡球焊接在PCB板上，由于焊点和PCB板的接触面积较大，所以内存芯片在运行中所产生的热量可以很容易地传导到PCB板上并散发出去。另外，CSP封装芯片采用中心引脚形式，有效地缩短了信号的传导距离，其衰减随之减少，芯片的抗干扰、抗噪性能也能得到大幅提升。

默认设置，令第1脚为圆形，其余脚为方形，以便区分。

(5) 单击next，进入轮廓宽度设置画面

(6) 单击next，进入焊盘间距设置画面
焊盘间距设置为0.5m，经计算行列间距为1.75mm。

(7) 单击next ，进入焊盘起始位置和命名方向设置
单击单选框可以确定焊盘起始位置，单击箭头可以改变焊盘命名方向。采用默认设置，将第一个焊盘设置在封装左上角，逆时针方向为命名方向。

(9) 单击next，进入封装命名

(10) 单击next，进入封装制作完成画面，单击Finish，退出封装导向

3.5 手工创建PCB元件不规则封装

(1) 打开PCB元件库NewPcbLib.PcbLib，执行Toois——>component Wizard，这时PCB Library操作界面的元件框会出现一个新的PCBCOMPONENT_1空文件，双击该文件，在弹出对话框中修改元件名称。
(2) 编辑工作环境设置
Tools——>Library Options，或工作区域单击右键，选择Library Options

(3) 放置焊盘
在Top Layer ——> P+P ,光标上会悬浮一个十字光标和一个焊盘，单击鼠标左键确定焊盘位置
(4) 编辑焊盘属性
双击焊盘即进入设置焊盘属性



(5) 绘制轮廓线
放置焊盘完毕后，需要绘制元件轮廓线。元件轮廓线：该元件封装在电路板上占据的空间大小，轮廓线的线形和大小取决于实际元件的形状和大小，通常需要测量实际元件。
(6) 绘制一段直线
单击Top Overlay(顶层覆盖) ,将活动层设置为顶层丝印层。P+L，光标将会变成十字形状，单击鼠标左键确定直线起点，并移动鼠标拉出一条直线，将直线 拉到合适的位置，单击鼠标左键确定直线终点，单击右键退出。

(7) 绘制一段弧线
P+（Arc(center)），光标变成十字形状，将光标移到坐标原点，单击鼠标左键确定圆心，然后将鼠标移动至直线的任一端，单击鼠标左键确定圆弧半径，再在直线两个端点两次单击鼠标左键确定圆弧

(8) Edit——>Set Reference 有三个选项Pin 1，Center，Location。可以选择合适的元件参考点。
