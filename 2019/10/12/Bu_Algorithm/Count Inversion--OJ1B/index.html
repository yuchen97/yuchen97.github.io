Count Inversion( OJ 1-B )

Problem Description

Recall the problem of finding the number of inversions. As in the course, we are given a sequence of $n$ numbers $a_1, a_2,\cdots,a_n$, and we define an inversion to be a pair $ia_j$.

We motivated the problem of counting inversions as a good measure of how different two orderings are. However, one might feel that this measure is too sensitive. Let’s call a pair a significant inversion if $i3a_j$. Give an $O(nlogn)$ algorithm to count the number of significant inversions between two orderings.

The array contains N elements (1<=N<=100,000). All elements are in the range from 1 to 1,000,000,000.

Input

The first line contains one integer N, indicating the size of the array. The second line contains N elements in the array.

  • 50% test cases guarantee that N < 1000.

    Output

    Output a single integer which is the number of pairs of significant inversions.

一般的逆序对是计算 $a_i>a_j$，而此题目是 $a_i > 3a_j$，为了减少内存空间，可以将 $a_i > 3a_j$转化成 $a_i - a_j > a_j +a_j$，另外由于数据比较大，所以数值类型使用的 long long int。

该算法主要是用了 Divide and Counqer 思想。选取一个中间点，分别对两边的子序列进行计算。算法内容可以结合下面的图文来进行理解。

一、Divide

假设有序列 $\{13, 8, 5, 4, 1, 2\}$，首先执行 Divide，将原序列分的不能再分时再执行 Conquer。
时间复杂度为 $T(n) = 2T(n/2)$.


二、Conquer

时间复杂度为 $T(n) = O(n)$.
总时间复杂度是 $T(n) = 2T(n/2) + O(n) = O(nlogn)$.

  • 第一次 Merge 时，$i=0, j=1, mid=0$ 针对 13 与 8，由于 13 大于 $3 \times 8$，$j-1 < mid + 1$，所以 $cnt == 0$，只将 13 和 8 做增序排序，并与原序列合并，$\{13, 8, 5, 4, 1, 2\}$->$\{8, 13, 5, 4, 1, 2\}$。
  • 第二次 Merge 时，$\{8, 13\}$ 和 $\{5\}$，首先比较 13是否大于 $3 \times 5$，不满足，不用比较 8 与 $3\times5$的大小，因为比8大的都没有 $3\times 5$大，所以此时 $cnt == 0$。$\{8, 13, 5, 4, 1, 2\}$->$\{5, 8, 13, 4, 1, 2\}$。
  • 第三次 Merge 时，$\{4\}$ 与 $\{1 \}$，$4 > 3\times 1$，$ ++cnt$。$\{5, 8, 13, 4, 1, 2 \}$->$\{5, 8, 13, 1, 4, 2\}$。
  • 第四次 Merge 时，$\{1, 4\}$ 与 $\{2 \}$，$4 < 3\times 2$，$ cnt$不变。$\{5, 8, 13, 1, 4, 2 \}$->$\{5, 8, 13, 1, 2, 4\}$。
  • 第五次 Merge 时，$\{5, 8, 13\}$ 与 $\{1, 2, 4 \}$，$13 < 3\times 1$，$ cnt$不变。再比较 13 与 2，因为最大的都没有4的三倍大，那其他的也就不用比了。$13 > 3\times 2$，此时 $cnt + 2$，因为右边除了4之外最大的都比13的三分之一小，所以其他的就不用比了。然后比较8 与 2，$cnt+2$。然后比较5 与 2，$5<3\times 4 2$，比较5与1，$cnt+1$。然后再进行排序 $\{5, 8, 13, 1, 2, \}$->$\{1, 2, 4, 5, 8, 13\}$。

最后逆序对一共有 6 对，分别是 (13, 1)、(13, 2)、(8, 2)、(8, 1)、(5, 1)、(4, 1).
